## **SECTION A — DEFINITIONS (WITH ANSWERS)**

---

### **1. Q: Define a computer.**

**A:** A computer is an electronic device that receives data, processes it according to instructions and produces output.

---

### **2. Q: Define data.**

**A:** Data is raw, unprocessed facts and figures entered into a computer.

---

### **3. Q: Define information.**

**A:** Information is processed, meaningful and organized data.

---

### **4. Q: What is an analogue computer?**

**A:** An analogue computer processes data in the form of continuous physical quantities such as voltage or temperature.

---

### **5. Q: What is a digital computer?**

**A:** A digital computer processes data represented in binary form (0s and 1s).

---

### **6. Q: Define hybrid computer.**

**A:** A hybrid computer combines features of both analogue and digital computers to handle continuous signals and digital data.

---

### **7. Q: What is speed in computers?**

**A:** Speed refers to how fast a computer can process instructions, usually measured in MIPS or GHz.

---

### **8. Q: What is accuracy in computers?**

**A:** Accuracy is the ability of a computer to perform operations without errors unless incorrect data or instructions are given.

---

### **9. Q: Define diligence.**

**A:** Diligence means a computer can work continuously for long periods without becoming tired or losing performance.

---

### **10. Q: What is storage capability?**

**A:** Storage capability refers to a computer’s ability to store large volumes of data both temporarily (primary memory) and permanently (secondary memory).

---

### **11. Q: Define versatility.**

**A:** Versatility means a computer can perform many different types of tasks, from calculations to multimedia.

---

### **12. Q: What is a vacuum tube?**

**A:** A vacuum tube is an electronic component used in 1st-generation computers to control electrical signals but consumed high power and generated heat.

---

### **13. Q: What is a transistor?**

**A:** A transistor is a semiconductor device used in 2nd-generation computers, smaller and more reliable than vacuum tubes.

---

### **14. Q: Define integrated circuit (IC).**

**A:** An IC is a small electronic chip containing multiple transistors and circuit elements, used in 3rd-generation computers.

---

### **15. Q: Define microprocessor.**

**A:** A microprocessor is a single-chip CPU used in 4th-generation computers that performs all arithmetic, logic and control operations.

---

### **16. Q: What is RAM?**

**A:** RAM (Random Access Memory) is volatile primary memory that temporarily stores data and instructions during processing.

---

### **17. Q: What is ROM?**

**A:** ROM (Read-Only Memory) is non-volatile memory that stores permanent data and instructions that cannot be modified easily.

---

### **18. Q: What is PROM?**

**A:** PROM (Programmable Read-Only Memory) is a type of ROM that can be programmed once by the user but cannot be erased.

---

### **19. Q: Define microcomputer.**

**A:** A microcomputer is a small, low-cost computer designed for individual use, built around a microprocessor.

---

### **20. Q: Define minicomputer.**

**A:** A minicomputer is a mid-sized computer capable of supporting multiple users simultaneously, used by small and medium businesses.

---

### **21. Q: Define mainframe.**

**A:** A mainframe is a large, powerful computer designed to handle massive data processing for large organizations.

---

### **22. Q: Define supercomputer.**

**A:** A supercomputer is the fastest and most powerful type of computer used for highly complex calculations like weather forecasting and simulations.

---

### **23. Q: What is AI in the 5th generation of computers?**

**A:** AI refers to computer systems designed to exhibit human-like intelligence, such as learning, reasoning and decision-making.

---

### **24. Q: Define software.**

**A:** Software is a collection of programs and instructions that tell the hardware how to operate.

---

### **25. Q: Define system software.**

**A:** System software manages and controls the hardware and provides a platform for running application software.

---

### **26. Q: Define application software.**

**A:** Application software consists of programs designed to perform specific user tasks such as word processing or accounting.

---

### **27. Q: Define hardware.**

**A:** Hardware refers to the physical components of a computer such as CPU, memory, keyboard and monitor.

---

### **28. Q: Define ALU.**

**A:** The Arithmetic Logic Unit (ALU) performs arithmetic operations (add, subtract, multiply) and logical operations (AND, OR, NOT).

---

### **29. Q: Define CU.**

**A:** The Control Unit (CU) directs and coordinates all operations of a computer by interpreting instructions.

---

### **30. Q: What is a register?**

**A:** A register is a small, high-speed storage location inside the CPU used to store temporary data and instructions.

---

### **31. Q: What is cache memory?**

**A:** Cache memory is a small, high-speed memory between the CPU and RAM that stores frequently used data to speed up processing.

---

### **32. Q: Define input device.**

**A:** An input device is hardware that allows users to enter data and instructions into a computer.

---

### **33. Define output device.**

**A:** An output device is hardware that displays or produces the processed results from the computer.

---

### **34. Q: Define IPO cycle.**

**A:** The Input-Process-Output cycle describes how a computer accepts input, processes it and produces output.

---

### **35. Q: Define CPU cycle.**

**A:** The CPU cycle consists of fetching, decoding, executing and storing instructions.

---

### **36. Q: What is MAR?**

**A:** The Memory Address Register holds the address of the memory location to be read or written.

---

### **37. Q: What is MBR/MDR?**

**A:** The Memory Buffer/Data Register holds the actual data that is being transferred to or from memory.

### **1. Q: Explain the characteristics of a computer in detail.**

**A:**  
A computer has several key characteristics:

1. **Speed:** Computers process millions of instructions per second due to electronic circuits.
    
2. **Accuracy:** When given correct data and instructions, computers produce error-free results.
    
3. **Diligence:** Computers do not get tired, bored or lose concentration; they can work continuously.
    
4. **Storage Capability:** Computers can store vast amounts of data in primary and secondary memory.
    
5. **Versatility:** Computers can perform various tasks such as calculations, gaming, design and communication.
    
6. **Automation:** After instructions are set, computers work automatically without user intervention.
    
7. **Consistency:** Output remains uniform and reliable regardless of workload.

### **2. Q: Describe the evolution of computers through different generations.**

**A:**  
Computer evolution is classified into five generations:

1. **First Generation (1940–1956):**
    
    - Used **vacuum tubes**
        
    - Very large, generated heat, high power consumption
        
    - Example: ENIAC, UNIVAC
        
2. **Second Generation (1956–1963):**
    
    - Used **transistors**
        
    - More reliable, smaller, faster than vacuum tubes
        
3. **Third Generation (1964–1971):**
    
    - Used **Integrated Circuits (ICs)**
        
    - Increased speed and efficiency; reduced cost and size
        
4. **Fourth Generation (1971–Present):**
    
    - Uses **microprocessors**
        
    - Birth of personal computers
        
    - Very high speed and storage
        
5. **Fifth Generation (Present & Future):**
    
    - Based on **Artificial Intelligence**, neural networks, robotics
        
    - Focus on learning, reasoning and self-decision

### **12. Q: Explain the computer booting process.**

**A:**  
Booting is the process of loading the operating system into memory. Steps:

1. Power is turned on.
    
2. BIOS performs POST (Power-On Self Test).
    
3. BIOS detects hardware and loads bootloader.
    
4. Bootloader loads OS kernel into memory.
    
5. OS initializes and system becomes ready.
    

Types:

- **Cold Booting:** Starting from power-off.
    
- **Warm Booting:** Restarting without cutting power.

### **29. Q: Explain in detail how the CPU executes an instruction (fetch–decode–execute cycle).**

**A:**  
CPU follows a cycle for each instruction:

1. **Fetch:**
    
    - Program Counter (PC) holds the address of next instruction.
        
    - Address is sent to MAR.
        
    - Data fetched into MDR/MBR.
        
    - Instruction loaded into Instruction Register (IR).
        
2. **Decode:**
    
    - Control Unit decodes instruction from IR.
        
    - Determines the operation and operands.
        
3. **Execute:**
    
    - ALU performs required arithmetic/logic operations.
        
    - Data is retrieved from memory or registers.
        
    - Results stored back in registers or memory.
        
4. **Store (if required):**
    
    - Results written back into memory.
        

This cycle repeats millions or billions of times per second.
