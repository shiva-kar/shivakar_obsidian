#### **1.1 Basic Computer Organisation**

Q1: What is a computer?

Answer: A computer is an electronic device that accepts data as input, processes it according to a set of instructions (program), stores the results, and provides information as output.

Q2: What is the Von Neumann Architecture?

Answer: It is a computer architecture design consisting of three main components: the Central Processing Unit (CPU), the Memory Unit, and the Input/Output (I/O) interfaces, all connected by a system bus.

Q4: What are the functional units of a computer?

Answer: Input Unit, Central Processing Unit (CPU), and Output Unit.

#### **1.2 Computer Hardware Components**

Q5: What is the CPU and what are its components?

Answer: The CPU (Central Processing Unit) is the "brain" of the computer responsible for processing instructions. Its components are:

1. **ALU (Arithmetic Logic Unit):** Performs math and logic operations.
    
2. **CU (Control Unit):** Directs the flow of data and instructions.
    
3. **Registers/Memory Unit:** Small, fast storage for immediate data.
    

Q6: What is the function of the ALU?

Answer: It performs arithmetic operations (addition, subtraction, multiplication, division) and logical operations (AND, OR, NOT, comparisons like <, >).

Q7: What is the function of the Control Unit (CU)?

Answer: It acts as the "traffic police" of the computer, decoding instructions and generating control signals to coordinate the activities of the ALU, memory, and I/O devices.

Q8: Differentiate between Primary and Secondary Memory.

Answer:

- **Primary Memory (RAM/ROM):** Volatile (loses data when power is off), faster, directly accessed by CPU, smaller capacity.
    
- **Secondary Memory (Hard Disk/SSD):** Non-volatile (retains data), slower, not directly accessed by CPU, larger capacity.
    

Q9: What is Cache Memory?

Answer: A very high-speed semiconductor memory which can speed up the CPU. It acts as a buffer between the CPU and the main memory.

Q10: List five Input Devices.

Answer: Keyboard, Mouse, Scanner, Microphone, Optical Character Reader (OCR).

Q11: List five Output Devices.

Answer: Monitor (VDU), Printer, Plotter, Speaker, Projector.

Q12: What is the difference between Hard Copy and Soft Copy?

Answer: Hard copy is a physical reproduction of data (e.g., printed paper), while soft copy is an electronic display of data on a screen.

#### **1.3 Types of Software**

Q13: Define Software.

Answer: A collection of instructions or programs that tell the hardware what to do.

Q14: What are the two main types of software?

Answer: System Software and Application Software.

Q15: What is System Software? Give examples.

Answer: Software designed to run the computer's hardware and application programs. Examples: Operating Systems (Windows, Linux), Device Drivers, Compilers.

Q16: What is Application Software? Give examples.

Answer: Software designed for the end-user to perform specific tasks. Examples: MS Word, Web Browsers, Games, Photoshop.

Q17: Differentiate between Compiler and Interpreter.

Answer:

- **Compiler:** Scans the whole program and translates it into machine code at once. It reports errors after scanning. Execution is faster. (Example: C).
    
- **Interpreter:** Translates the program line-by-line. It stops at the first error. Execution is slower. (Example: Python).
    

Q18: What is an Assembler?

Answer: A utility program that translates Assembly Language (low-level mnemonic code) into Machine Code (binary).

Q19: What is a Linker?

Answer: A system program that takes one or more object files generated by a compiler and combines them into a single executable file.

Q20: What is a Loader?

Answer: A part of the operating system that is responsible for loading programs and libraries from the hard disk into the RAM so they can be executed.

#### **1.4 Operating System Concepts**

Q21: What is an Operating System (OS)?

Answer: System software that acts as an interface between the user and the computer hardware. It manages system resources.

Q22: List the primary functions of an Operating System.

Answer:

1. **Process Management:** Scheduling tasks.
    
2. **Memory Management:** Allocating RAM.
    
3. **File Management:** Organizing files on storage.
    
4. **Device Management:** Controlling peripherals.
    
5. **Security:** Protecting data.
    

Q23: What is a Batch Operating System?

Answer: An OS where similar jobs are grouped together into batches and executed one after another without user intervention.

Q24: What is a Time-Sharing Operating System?

Answer: An OS that allows multiple users to share resources simultaneously by switching the CPU between them very quickly (multitasking).

Q25: What is a Real-Time Operating System (RTOS)?

Answer: An OS intended to serve real-time applications that process data as it comes in, typically without buffer delays (e.g., air traffic control systems).

#### **1.5 Case Study**

Q26: What is Linux?

Answer: An open-source, Unix-like operating system kernel. It is known for its stability, security, and use in servers and embedded systems.

Q27: What is the core component of Linux called?

Answer: The Kernel.

### **Module 2: Programming in C**

#### **2.1 Introduction to C**

Q28: Who developed C and when?

Answer: Dennis Ritchie in 1972 at AT&T Bell Laboratories.

Q29: Why is C called a "Middle-Level" language?

Answer: Because it combines features of high-level languages (easy syntax, portability) with the low-level features of assembly language (direct memory manipulation via pointers).

Q30: Explain the basic structure of a C program.

Answer:

1. **Documentation Section:** Comments.
    
2. **Link Section:** Header files (`#include`).
    
3. **Definition Section:** Macros (`#define`).
    
4. **Global Declaration:** Global variables.
    
5. **`main()` Function:** Execution start point.
    
6. **Subprograms:** User-defined functions.
    

#### **2.2 C Statements & Data Types**

Q31: What is a Variable?

Answer: A named storage location in memory used to hold a value that can change during program execution.

Q32: What are the rules for naming identifiers (Variables) in C?

Answer:

1. Must start with a letter or underscore.
    
2. Can contain letters, digits, and underscores.
    
3. Cannot contain spaces or special symbols.
    
4. Cannot be a keyword.
    
5. Case-sensitive.
    

Q33: What is a Constant? List types.

Answer: A fixed value that does not change during execution. Types: Integer constants, Floating-point constants, Character constants, String constants.

Q34: What is a Keyword? Give examples.

Answer: Reserved words with predefined meanings in the compiler. Examples: int, return, if, while, void.

Q35: Explain the fundamental Data Types in C.

Answer:

1. **`int`:** Integers (2 or 4 bytes). Format: `%d`.
    
2. **`char`:** Characters (1 byte). Format: `%c`.
    
3. **`float`:** Single-precision decimal (4 bytes). Format: `%f`.
    
4. **`double`:** Double-precision decimal (8 bytes). Format: `%lf`.
    
5. **`void`:** Empty/no value.
    

Q36: What is Conditional Compilation?

Answer: Instructions to the preprocessor (like #ifdef, #ifndef, #else, #endif) to compile specific parts of the code only if certain conditions are met.

Q37: What is #define used for?

Answer: It is a preprocessor directive used to define macros or symbolic constants (e.g., #define PI 3.14).

#### **2.3 Operators**

Q38: List the categories of Operators in C.

Answer: Arithmetic, Relational, Logical, Assignment, Increment/Decrement, Conditional (Ternary), Bitwise, Special (comma, sizeof).

Q39: What is the Modulus Operator (%)?

Answer: An arithmetic operator that returns the remainder of a division. It works only with integers.

Q40: Explain the difference between == and =.

Answer: = is the Assignment operator (assigns value). == is the Relational operator (checks equality).

Q41: Explain Logical Operators (&&, ||, !).

Answer:

- `&&` (AND): True only if both conditions are true.
    
- `||` (OR): True if at least one condition is true.
    
- `!` (NOT): Reverses the truth value.
    

Q42: What are Bitwise Operators?

Answer: Operators that manipulate individual bits of data. Examples: & (AND), | (OR), ^ (XOR), ~ (NOT), << (Left Shift), >> (Right Shift).

Q43: Explain Pre-increment (++i) vs. Post-increment (i++).

Answer:

- **Pre-increment (`++i`):** Increments the value first, then uses it.
    
- **Post-increment (`i++`):** Uses the current value first, then increments it.
    

Q44: What is the Ternary Operator?

Answer: A shorthand for if-else. Syntax: (Condition) ? True_Value : False_Value;.

#### **2.4 Input/Output**

Q45: What is the difference between printf() and scanf()?

Answer: printf() sends formatted output to the screen. scanf() reads formatted input from the keyboard.

Q46: Why do we use & in scanf?

Answer: To provide the memory address of the variable where the input value should be stored.

Q47: Differentiate between Formatted and Unformatted I/O.

Answer:

- **Formatted:** Allows control over the data format (e.g., `printf`, `scanf`).
    
- **Unformatted:** Reads/writes raw characters or strings without format specifiers (e.g., `getch`, `putch`, `gets`, `puts`).
    

Q48: What do sprintf and sscanf do?

Answer:

- **`sprintf`:** Prints formatted output into a string (buffer) instead of the console.
    
- **`sscanf`:** Reads formatted input from a string instead of the keyboard.

### **Module 3: Decision Making in C**

#### **3.1 Decision Statements**

Q49: Explain the if statement.

Answer: A control statement that executes a block of code only if a specified condition evaluates to true.

Q50: What is the "Dangling Else" problem?

Answer: An ambiguity in nested if statements where it is unclear which if an else belongs to. In C, an else always pairs with the nearest unmatched if.

Q51: Explain the switch statement.

Answer: A multi-way branch statement that tests a variable against a list of values (cases).

Q52: What are the restrictions of a switch case expression?

Answer: The expression must result in an integer or character type. Floats and strings are not allowed.

Q53: What is the purpose of the break statement in a switch?

Answer: To exit the switch block immediately. Without it, execution "falls through" to the next case.

Q54: What is the default case?

Answer: An optional case in a switch statement that runs if no other case matches.

#### **3.2 Loops**

Q55: What are the three types of loops in C?

Answer: for loop, while loop, do-while loop.

Q56: Differentiate between while and do-while loops.

Answer:

- **`while`:** Entry-controlled. Checks condition first. Might run 0 times.
    
- **`do-while`:** Exit-controlled. Checks condition last. Guaranteed to run at least 1 time.
    

Q57: What is the syntax of a for loop?

Answer: for (initialization; condition; increment/decrement) { // code }

Q58: What is a Nested Loop?

Answer: A loop inside another loop (e.g., using two loops to print a matrix or a pattern).

Q59: Differentiate between break and continue.

Answer:

- **`break`:** Terminates the loop entirely.
    
- **`continue`:** Skips the rest of the current iteration and jumps to the next iteration.
    

#### **3.3 Command Line Arguments**

Q60: What are Command Line Arguments?

Answer: Arguments passed to the main() function from the command line shell when the program is executed.

Q61: What are argc and argv?

Answer:

- **`argc` (Argument Count):** The number of arguments passed (includes the program name).
    
- **`argv` (Argument Vector):** An array of character pointers (strings) containing the arguments.
    

#### **3.4 Storage Classes**

Q62: List the four Storage Classes in C.

Answer: auto, extern, static, register.

Q63: Explain the auto storage class.

Answer: The default for local variables. Stored in stack. Garbage initial value. Scope is local to the block.

Q64: Explain the static storage class.

Answer: Preserves value between function calls. Initialized to zero by default. Stored in RAM. Scope is local but lifetime is global.

Q65: Explain the extern storage class.

Answer: Used to declare a global variable that is defined in another file. It extends the visibility of the variable.

Q66: Explain the register storage class.

Answer: Requests the compiler to store the variable in a CPU register for faster access. Used for loop counters. Cannot use & operator on it.

### **Module 4: Arrays and Functions**

#### **4.1 Arrays**

Q67: What is an Array?

Answer: A collection of elements of the same data type stored in contiguous memory locations.

Q68: How do you declare an array?

Answer: DataType ArrayName[Size]; (e.g., int marks[5];)

Q69: What is the index of the first element in an array?

Answer: 0.

Q70: How is a 2D array declared?

Answer: int matrix[row][col]; (e.g., int matrix[3][3];)

Q71: How are arrays stored in memory?

Answer: In sequential memory addresses. For a 2D array, C uses "Row-Major" order (row by row).

#### **4.2 Functions**

Q72: What is a Function?

Answer: A self-contained block of code designed to perform a specific task.

Q73: What is a Function Prototype?

Answer: A declaration that tells the compiler about the function's name, return type, and parameters before it is defined.

Q74: Differentiate between Actual and Formal Arguments.

Answer:

- **Actual:** The values passed during the function call.
    
- **Formal:** The variables in the function definition that receive the values.
    

Q75: Explain Call by Value.

Answer: The value of the variable is passed to the function. Changes made inside the function do not affect the original variable.

Q76: Explain Call by Reference.

Answer: The address of the variable is passed (using pointers). Changes made inside the function affect the original variable.

Q77: What is Recursion?

Answer: A process where a function calls itself. It must have a base case to stop the infinite loop.

#### **4.3 String Handling**

Q78: What is a String in C?

Answer: An array of characters terminated by a null character (\0).

Q79: List common String Handling functions from <string.h>.

Answer:

- `strlen(s)`: Returns length.
    
- `strcpy(d, s)`: Copies source to destination.
    
- `strcat(d, s)`: Concatenates (joins) source to destination.
    
- `strcmp(s1, s2)`: Compares two strings.
    
- `strrev(s)`: Reverses a string.
    

Q80: What does strcmp return?

Answer:

- 0 if strings are equal.
    
- Negative value if s1 < s2 (ASCII).
    
- Positive value if s1 > s2.
    

Q81: How do you pass an array to a function?

Answer: By passing the name of the array. The name of the array acts as a pointer to the first element (Base Address).

### **Module 5: Advanced Features in C**

#### **5.1 Pointers**

Q82: What is a Pointer?

Answer: A variable that stores the memory address of another variable.

Q83: Explain the & and * operators.

Answer:

- **`&` (Address-of):** Returns the address of a variable.
    
- **`*` (Dereference):** Returns the value stored at the address.
    

Q84: What is Pointer Arithmetic?

Answer: Operations performed on pointers. Adding 1 to a pointer increases the address by the size of the data type it points to (e.g., +4 bytes for int).

Q85: What is a Dangling Pointer?

Answer: A pointer that points to a memory location that has been deleted or freed.

Q86: What is a Null Pointer?

Answer: A pointer that is assigned the value NULL (0), indicating it points to nothing.

Q87: What is a void pointer?

Answer: A generic pointer that can point to any data type. It cannot be dereferenced directly without casting.

Q88: What is an Array of Pointers?

Answer: An array where each element is a pointer. (e.g., int *arr[5]).

#### **5.2 Structures**

Q89: What is a Structure?

Answer: A user-defined data type that groups variables of different data types under a single name.

Q90: What is the syntax to declare a structure?

Answer:

C

```
struct Student {
    int id;
    char name[20];
};
```

Q91: How do you access structure members?

Answer: Using the dot operator (.). Example: s1.id = 10;.

Q92: What is typedef?

Answer: A keyword used to give a new name (alias) to an existing data type. Often used with structures to simplify syntax.

Q93: What is an Arrow Operator (->)?

Answer: Used to access structure members when using a pointer to a structure.

#### **5.3 Unions**

Q94: What is a Union?

Answer: A user-defined data type similar to a structure, but all members share the same memory location.

Q95: What is the main difference between Structure and Union?

Answer:

- **Structure:** Allocates memory for _all_ members (Size = sum of all members).
    
- **Union:** Allocates memory for only the _largest_ member (Size = size of largest member).
    

Q96: When would you use a Union?

Answer: When you need to store different types of data in the same variable but only one at a time, to save memory.

#### **5.4 File Handling**

Q97: What is a File Pointer?

Answer: A pointer of type FILE * used to handle files.

Q98: List the modes for opening a file using fopen.

Answer:

- `"r"`: Read.
    
- `"w"`: Write (Overwrites or creates).
    
- `"a"`: Append (Adds to end).
    
- `"r+"`: Read and Write.
    

Q99: What does fopen return if the file cannot be opened?

Answer: NULL.

Q100: List common File Handling functions.

Answer:

- `fopen()`: Opens file.
    
- `fclose()`: Closes file.
    
- `fprintf()`: Writes formatted data.
    
- `fscanf()`: Reads formatted data.
    
- `fgetc()`: Reads a character.
    
- `fputc()`: Writes a character.
    

Q101: What is EOF?

Answer: End Of File. A special value returned by file reading functions when the end of the file is reached.

#### **5.5 Transition from C to C++**

Q102: What is the main difference between C and C++?

Answer: C is a Procedural Oriented Programming (POP) language, while C++ is an Object-Oriented Programming (OOP) language.

Q103: What are the key features of OOP introduced in C++?

Answer:

1. **Classes and Objects.**
    
2. **Encapsulation.**
    
3. **Inheritance.**
    
4. **Polymorphism.**
    
5. **Abstraction.**
    

Q104: How does I/O differ in C++?

Answer: C uses printf/scanf (functions). C++ uses cout/cin (objects).

#### **Dynamic Memory Allocation**

Q105: What is Dynamic Memory Allocation?

Answer: Allocating memory manually during runtime (while the program is running) rather than compile time.

Q106: List the functions for Dynamic Memory Allocation.

Answer: malloc(), calloc(), realloc(), free().

Q107: Differentiate between malloc and calloc.

Answer:

- **`malloc`:** Allocates a single block of memory. Contains garbage values.
    
- **`calloc`:** Allocates multiple blocks of memory (for arrays). Initializes memory to zero.
    

Q108: What is free()?

Answer: A function used to deallocate (release) the memory allocated by malloc/calloc to prevent memory leaks.

Q109: What is a Memory Leak?

Answer: When a program allocates memory but fails to free it back to the system after use.

#### **Advanced Pointer Concepts**

Q110: What is a Pointer to a Pointer?

Answer: A pointer variable that stores the address of another pointer. Syntax: int **p;.

Q111: What is a Function Pointer?

Answer: A pointer that points to the code of a function in memory. It can be used to pass functions as arguments to other functions.

#### **Common Coding/Output Questions (Logic)**

Q112: What is the output of printf("%d", 10/3);?

Answer: 3 (Integer division truncates the decimal).

Q113: What is the output of printf("%f", 10.0/3);?

Answer: 3.333333 (Floating point division).

Q114: If int a[5] = {1, 2};, what is a[4]?

Answer: 0 (Partial initialization sets remaining elements to zero).

Q115: What is the value of x after x = 5; y = x++;?

Answer: x is 6, y is 5 (Post-increment).

Q116: What is the value of x after x = 5; y = ++x;?

Answer: x is 6, y is 6 (Pre-increment).

Q117: What is an infinite loop? Give an example.

Answer: A loop that never terminates. Example: while(1) { printf("Hello"); }.

Q118: How do you access the last element of an array of size n?

Answer: array[n-1].

Q119: Can a function return more than one value?

Answer: No, not directly. However, it can return multiple values by using Pointers (Call by Reference) or by returning a Structure.

Q120: What is the size of a pointer in C?

Answer: It depends on the architecture. 2 bytes in 16-bit, 4 bytes in 32-bit, and 8 bytes in 64-bit systems. It is independent of the data type it points to.

Q152: Can a switch statement evaluate a float value?

Q166: Why is gets() considered dangerous?

**Answer:** It does not check for buffer overflows; it reads characters until a newline is found, potentially overwriting memory outside the array45.

Q167: What function should be used instead of gets() for safety?

**Answer:** `fgets()`46.

Q168: What does the strcat() function do?

**Answer:** It concatenates (appends) one string to the end of another47.

Q169: What is the difference between strcmp() and strncmp()?

**Answer:** `strcmp()` compares two entire strings, while `strncmp()` compares only the first `n` characters of the strings48.

### **Module 5: Pointers, Structures, & Files**

#### **Pointers**

Q170: What is a Pointer?

**Answer:** A variable that stores the memory address of another variable49.

Q171: What is the Indirection Operator?

**Answer:** The `*` operator, which accesses the value at the address held by a pointer50.

Q172: What is Pointer Arithmetic?

Answer: Performing addition or subtraction on pointers. Incrementing a pointer increases its address by the size of the data type it points to51.

Q173: What is a "Void Pointer" (void *)?

Answer: A generic pointer that can point to any data type. It cannot be dereferenced directly without casting52.

Q174: What is a "Dangling Pointer"?

**Answer:** A pointer that points to a memory location that has been deleted (freed) or is no longer valid53.

Q175: How do you fix a dangling pointer?

**Answer:** Assign `NULL` to the pointer after freeing the memory54.

#### **Structures & Unions**

Q176: What is a Structure?

**Answer:** A user-defined data type that groups related variables of different data types under a single name55.

Q177: How do you access a member of a structure?

**Answer:** Using the dot operator (`.`) (e.g., `student.name`)56.

Q178: What is the arrow operator (->) used for?

**Answer:** To access structure members using a pointer to the structure (e.g., `ptr->name`)57.

Q179: What is the main difference between a Structure and a Union regarding memory?

Answer: In a structure, each member has its own memory space. In a union, all members share the same memory location, so the size is equal to the largest member58.

Q180: When would you use a Union?

**Answer:** When you need to work with different data types but only one at a time, specifically to save memory59.

#### **File Handling**

Q181: What is the FILE data type?

**Answer:** A structure defined in `<stdio.h>` that holds information about a file stream60.

Q182: What does the mode "w" do in fopen()?

Answer: It opens a file for writing. If the file exists, it overwrites it. If not, it creates a new one61616161.

Q183: What does the mode "a" do in fopen()?

Answer: It opens a file for appending. Data is added to the end of the file. It creates the file if it doesn't exist62626262.

Q184: What is EOF?

Answer: End Of File. It is a special value (usually -1) returned by input functions when no more data can be read from a file63.

Q185: What is the difference between text files and binary files?

Answer: Text files store data as human-readable characters (ASCII). Binary files store data in the same format as computer memory (0s and 1s), which is compact but not human-readable 64.

Q186: Which function is used to write a block of data (like an array or structure) to a file?

**Answer:** `fwrite()`65.

Q187: Which function is used to read a block of data from a file?

**Answer:** `fread()`66.

Q188: Why should you always close a file using fclose()?

**Answer:** To release system resources, flush buffers (ensure data is written), and prevent data corruption67.

Q190: List three features of C++ not present in C.
