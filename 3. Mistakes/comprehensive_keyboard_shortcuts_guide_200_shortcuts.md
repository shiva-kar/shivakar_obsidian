## 🧭 Basic Universal Shortcuts

|Shortcut Keys|Function / Description|Example Use Case|
|---|---|---|
|**Ctrl + C**|Copy|Copy selected text or files|
|**Ctrl + X**|Cut|Cut text or move files|
|**Ctrl + V**|Paste|Paste copied text or items|
|**Ctrl + Z**|Undo|Undo last action|
|**Ctrl + Y**|Redo|Redo an undone action|
|**Ctrl + A**|Select All|Select all text or files|
|**Ctrl + S**|Save|Save document or file|
|**Ctrl + O**|Open|Open a file or document|
|**Ctrl + N**|New|Create a new file, document, or window|
|**Ctrl + P**|Print|Open print dialog|
|**Alt + F4**|Close Window|Close active window or application|
|**Alt + Tab**|Switch Between Apps|Quickly change between open programs|
|**Ctrl + Shift + Esc**|Open Task Manager|Open Windows Task Manager directly|
|**Ctrl + Esc**|Open Start Menu|Show Windows Start Menu|
|**F1**|Help|Open help menu in most programs|
|**F2**|Rename|Rename a selected file or folder|
|**F3**|Search|Open search feature|
|**F5**|Refresh|Refresh page or reload folder|
|**F11**|Full Screen|Toggle fullscreen mode in browser or Explorer|
|**Shift + Delete**|Permanent Delete|Delete item permanently without Recycle Bin|
|**Alt + Enter**|Properties|View properties of selected file or folder|
