## 47. Debugging & Diagnostics

- Compile with `-g -O0 -Wall -Wextra`.
    
- Use sanitizers:
    
    - AddressSanitizer (`-fsanitize=address`)
        
    - UndefinedBehaviorSanitizer (`-fsanitize=undefined`)
        
- Logging frameworks: spdlog, glog
    
- Assertions for runtime checks.
